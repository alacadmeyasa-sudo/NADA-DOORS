# معرض نادا دورز — صفحة هبوط فرع نجران 🚪

قالب صفحة هبوط احترافي (RTL) لمعرض **نادا دورز - فرع نجران**، قابل للتعديل بالكامل من خلال **Elementor**، مع زر **واتساب مميّز** يرسل رسالة جاهزة توضّح أن التواصل تمّ عبر صفحة فرع نجران.

---

## 📦 محتويات المشروع

| الملف | الوصف |
|------|--------|
| `template/nada-doors-najran-elementor.json` | قالب Elementor جاهز للاستيراد والتعديل |

## 🚀 طريقة استيراد القالب في WordPress + Elementor

1. سجّل الدخول إلى لوحة تحكم ووردبريس، وتأكّد من تفعيل إضافة **Elementor**.
2. من القائمة الجانبية اذهب إلى: **Templates ← Saved Templates** (القوالب ← القوالب المحفوظة).
3. اضغط زر **Import Templates** (استيراد القوالب) بالأعلى.
4. اختر الملف: `template/nada-doors-najran-elementor.json` ثم **Import Now**.
5. أنشئ صفحة جديدة (**Pages ← Add New**)، اضغط **Edit with Elementor**.
6. من أيقونة المجلد 📁 داخل المحرّر، افتح **My Templates** واختر القالب المستورد ثم **Insert**.
7. عدّل النصوص والصور والألوان كما تحب — كل عنصر قابل للتعديل بالسحب والإفلات.
8. اضغط **Publish / Update** للنشر.

> 💡 يُفضّل ضبط تخطيط الصفحة على **Elementor Canvas** (من إعدادات الصفحة) للحصول على شكل صفحة الهبوط الكامل بدون رأس/تذييل القالب.

## ✏️ تعديل البيانات بسرعة

لتغيير الهاتف أو النصوص أو أنواع الأبواب، عدّل القيم في أعلى ملف `build_template.py` ثم أعد التوليد:

```bash
python3 build_template.py
```

أهم المتغيّرات:
- `PHONE_INTL` و `PHONE_LOCAL` — رقم الهاتف.
- `WA_MESSAGE` — نص رسالة الواتساب الترحيبية.
- `PRODUCTS` و `FEATURES` — قوائم المنتجات والمميزات.
- `C_GOLD` / `C_DARK` … — ألوان الهوية.

> ملاحظة: كل تعديلات الواجهة يمكن إجراؤها مباشرة داخل Elementor بدون لمس الكود إطلاقاً.
